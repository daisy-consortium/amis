Building the AMIS installer
---
Tools required:

1. NSIS: http://nsis.sf.net

2. Plugins:

	a. version.dll: For getting the windows version used
	http://nsis.sourceforge.net/Version_plug-in
	b. inetc.dll: For downloading files
	http://nsis.sourceforge.net/Inetc_plug-in
	
Useful editors (not required though):

1. Eclipse plug-in: http://nsis.sourceforge.net/EclipseNSIS_-_NSIS_plugin_for_Eclipse

2. HM NIS Edit: http://nsis.sourceforge.net/HM_NIS_Edit
